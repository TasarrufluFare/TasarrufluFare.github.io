Fare Color Grader

Created by NEZİH BORA YAVAŞ

APPLY COLOR GRADING OPERATIONS AUTOMATICALLY

Using artificial neural networks, you can perfom various color grading operations, which aims to provide a realistic balance with a single click.

EASILY FINE-TUNE COLORS OR USE BUILD-IN PRESETS

You can fine-tune the automatic color grading operation or build an operation from scratch. You will always have build-in color presets besides you can save your own presets.

Match Color With A Reference Image

You can match you images color scale with a reference image given by you. It works better for environmental light operations.

WORK WITH NODES

You can activate or deactive any specific effect on your image using nodes. Also with nodes menu you can edit you images in more effective way.

General

Nezih Bora Yavaş's self-paced project, Fare Color Grader, attempts to be an alternative to industry lead color grading applications. Fare Color Grader is still under development and continues to gain new features.

Curves
